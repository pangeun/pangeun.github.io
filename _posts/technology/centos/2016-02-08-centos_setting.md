호스트 설정
==================
```
/etc/hosts
```

호스트네임 설정
==================
```
/etc/sysconfig/network
```

네임서버 설정
==================
- nameserver를 구글로 변경한다.
```
/etc/resolv.conf
```

랜카드 설정
==================
```
/etc/sysconfig/network-scripts/ifcfg-eth0
```

루프백 설정
==================
```
/etc/sysconfig/network-scripts/ifcfg-lo
```
